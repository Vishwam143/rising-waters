1. Introduction & Project Objectives

Flooding is a catastrophic hydrological event characterized by the overflow of water onto land that is normally dry. Meteorological parameters, such as rainfall, cloud cover, humidity, and temperature, directly dictate these events. The primary objective of this project is to develop a predictive engine that handles multi-dimensional weather data, conducts exploratory data analysis, trains supervised ML models, and deploys the best performing classifier.

Core Objectives:
  • Predictive Accuracy: Establish early flood warning notifications using robust algorithms, targeting >95% classification performance.
  • Multi-Model Comparison: Train and compare Decision Trees, Random Forests, K-Nearest Neighbors, and XGBoost classifiers.
  • Serialized Integration: Expose predictions via a lightweight Flask web interface, making it easily wrap-able for cloud scaling.

2. Existing vs Proposed System

Existing System

Traditional systems rely on physical hydrological models and radar imagery. These processes are computationally heavy, struggle with non-linear weather interfaces, and require localized manual adjustments, leading to latency in disaster notification alerts.

Proposed System

A supervised machine learning predictive pipeline. Preprocesses raw environmental readings, scales them through standard distributions, and runs inferences on highly optimized boosted decision trees (XGBoost). Achieves near-instant predictions (<10ms).

5. Methodology & Algorithms

To build a high-performance system, four machine learning algorithms were trained and comparatively evaluated:

Decision Tree Classifier

Splits the dataset based on feature thresholds, creating tree nodes that maximize information gain (Gini impurity / Entropy). Quick but prone to overfitting.

K-Nearest Neighbors (KNN)

Classifies a sample based on the majority label of its k closest neighbors in Euclidean distance. Computationally heavy at test time.

Random Forest Classifier

An ensemble bagging algorithm that trains multiple Decision Trees in parallel on bootstrapped subsets, taking majority votes to reduce variance.

XGBoost (Extreme Gradient Boosting)

An advanced gradient boosting framework that trains weak learners sequentially, optimizing loss functions with regularization to prevent overfitting. Achieved 96.55% accuracy.
